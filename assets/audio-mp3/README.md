# MP3 Web Copies

Generated MP3 files live here, preserving the same folders as `assets/audio/`.

```text
assets/audio/homepage music/Song.flac
assets/audio-mp3/homepage music/Song.mp3
```

Run this from the project root to create or update MP3 copies:

```bash
npm run convert:mp3
```

The original FLAC files stay in `assets/audio/`.
